 # Landing Page

## About

This project is a personal portfolio landing page created as part of The Odin Project. The objective is to design a landing page that closely resembles the requirements provided by The Odin Project. This project serves as a practical exercise to enhance HTML and CSS skills by constructing a personalized portfolio landing page from scratch.
